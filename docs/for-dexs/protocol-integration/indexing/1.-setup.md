# 1. Setup

### Install Rust

Install <a href="https://www.rust-lang.org/tools/install" target="_blank" rel="noopener noreferrer">Rust</a>. You can do so with the following command:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Install Substreams

You can do so with any of the following:

#### Using Homebrew:

```bash
brew install streamingfast/tap/substreams
```

#### Using precompiled binaries

```bash
# Use correct binary for your platform
LINK=$(curl -s https://api.github.com/repos/streamingfast/substreams/releases/latest | awk '/download.url.*linux/ {print $2}' | sed 's/"//g')
curl -L  $LINK  | tar zxf -
```

#### Compiling from source:

```bash
git clone https://github.com/streamingfast/substreams
cd substreams
go install -v ./cmd/substreams
```

### Install Buf

#### Using Homebrew:

```bash
brew install bufbuild/buf/buf
```

For other installation methods, see the <a href="https://buf.build/docs/installation/#__tabbed_1_1" target="_blank" rel="noopener noreferrer">official buf website</a>\

### Fork the monorepo

1. Start by making a fork of the <a href="https://github.com/propeller-heads/tycho-indexer" target="_blank" rel="noopener noreferrer">Tycho monorepo</a> repository
2. Clone the fork you just created
3. Make sure everything compiles fine

   ```bash
   cd protocols/substreams
   cargo check --all
   ```
