# tycho

Meta-crate providing a compatible, versioned set of [Tycho](https://github.com/propeller-heads/tycho) ecosystem crates.

Add this crate to your dependencies to get a coherent set of `tycho-client`, `tycho-common`, `tycho-simulation`, and `tycho-execution` crates that are guaranteed to work together.

## Features

- `evm` *(default)* — includes `tycho-simulation` and `tycho-execution` with EVM support
- `rfq` — adds RFQ support on top of `evm`
